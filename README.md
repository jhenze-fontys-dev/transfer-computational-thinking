## Abstract

This study investigates the role of **explicit attention to transfer** in the cross-curricular applicability of **Computational Thinking (CT)** within Dutch secondary education.

Recent curriculum developments position Computational Thinking explicitly within the lower grades of secondary education, while later curriculum documents increasingly assume that these thinking strategies will naturally transfer to other subject domains. This research examines whether that assumption is supported by educational theory and classroom observations.

A digital learning environment was designed in which Python is used as a **didactical tool** rather than as the learning objective itself. Within this environment, an authentic classroom study investigated whether reducing programming complexity through a custom Python module supports students in applying Computational Thinking during a subject-specific task.

The results indicate that reducing technical complexity makes programming activities more accessible, but **does not automatically lead to visible Computational Thinking**. The findings suggest that transfer requires explicit educational support and should not be assumed to emerge solely through programming activities.
